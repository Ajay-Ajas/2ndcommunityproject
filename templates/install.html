⚠ Installation & Safety Guide

Power Safety
• Turn OFF the main power before installation
• Never touch exposed conductors
• Use insulated tools if required

Sensor Installation
• Clamp the CT sensor on the main phase wire only
• Do NOT clamp both wires together
• Ensure secure attachment & insulation

Device Setup
• Restore power after installation
• Ensure ESP32 device is powered ON
• Confirm WiFi connection is active

System Activation
After installation, press "I Have Connected It".
The system will verify electrical flow and activate monitoring.
